## O que é o NPSharp

**NPSharp** é um editor de código leve, rápido e extensível.  
Combina simplicidade com ferramentas profissionais para o ciclo completo de desenvolvimento:

- edição avançada
- navegação inteligente
- debugging leve
- extensões poderosas
- integração com ferramentas externas

Atualizações frequentes trazem novas features e correções.

## Extensões Incluídas

O NPSharp inclui extensões embutidas para várias linguagens.

Extensões com suporte profundo usam o sufixo:

<language>-language-features

yaml
Copiar código

Exemplo:

- `json` → syntax highlight
- `json-language-features` → IntelliSense, Go to Definition, etc.
